Patient Tracking & Scheduling System

This project is a simple web-based prototype designed to demonstrate patient tracking and appointment scheduling functionality in a healthcare environment.

Features
  Add patient information
  View patient list
  Schedule appointments
  View scheduled appointments
  Data stored using browser local storage

How to Run

  1. Open the 1_code folder
  2. Open index.html in a web browser

 Project Structure
  1_code → Application files 
  2_unit_testing → Unit testing documentation
  3_integration_testing → Integration testing documentation
  4_data_collection → Data handling description
  5_documentation → Final reports and project documents
